search_for_path uses an A* search to find a path from the start to goal poses. The search assumes a circular robot

Parameters
startStarting pose of the robot
goalDesired goal pose of the robot
distancesDistance to the nearest obstacle for each cell in the grid
paramsParameters specifying the behavior of the A* search
Returns
The path found to the goal, if one exists. If the goal is unreachable, then a path with just the initial pose is returned, per the path2D_t specification.
