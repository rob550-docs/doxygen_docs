Detailed Description

MotionPlannerServer is a simple ActionServer(ROS-esque) that will serve as a way from the MBot webapp to interact with the MBot Motion Planner. The sole purpose of MotionPlannerServer is to receive a request to plan from the MBot webapp (when the user clicks a goal location), use the MBot motion planner to generate a plan, and finally publish the planned path.
