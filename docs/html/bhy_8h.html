write_my_bus_xy(u8 device_addr, u8 register_addr, u8 * data, u8 length);

The BHY_WR_FUNC_PTR would equal:

BHY_WR_FUNC_PTR s8 (* bus_write)(u8, u8, u8 *, u8)
