Detailed Description

Provides interface to the BMP280 barometer sensor on the MBot robot system using I2C.

>

This code handles initialization, reading, and resetting of the BMP280 sensor, as well as conversion of fetched raw sensor data to actual temperature and pressure values using the calibration parameters.

Sensor's default I2C address is 0x76 and required calibration parameters are read from registers starting at address 0x88 to 0x9F.

Functions mbot_initialize_barometer, mbot_read_barometer_pressure, and mbot_read_barometer_temperature use the BMP280 sensor functions to allow easy access of the barometer data.
