inline

angle_diff_abs_pi_2 finds the difference in radians between two angles where the range is [0, PI/2]. The calculation is the angle_diff. If the abs(diff) > M_PI/2, then the result is PI - abs(diff). This function is intended for cases where the angles represent line slopes rather than vectors. The slope only can differ by at most PI/2.

Parameters
lhsAngle on left of '-'
rhsAngle on right of '-'
Returns
lhs - rhs, in the range [0, PI/2].
