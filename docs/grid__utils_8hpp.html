const Grid &  grid

global_position_to_grid_cell converts a point in the global coordinate system to a cell in the grid coordinate system.

The global point likely falls somewhere within a cell in the grid. This function truncates the calculation to return the origin of the cell in which the point falls. If fractions of a cell are important – say for ray tracing – then global_position_to_grid_position should be called instead, as it allows for fractions of a cell.
